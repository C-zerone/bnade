# BNADE

## 项目介绍
BNADE全称Battle Net Auction Data Extracting, 网站www.bnade.com用于魔兽世界拍卖行相关数据的查询和分析，项目包括web的前后端以及后台爬虫程序

## 开发环境
- 开发语言: Java 1.8
- 使用的框架： Jersey, HightChart, Bootstrap, JQuery
- 数据库： MySQL
- Web容器： Tomcat7
- 构建工具：Maven, gulp
